## Overview

This repository contains structured notes created while studying Microsoft Defender XDR and Microsoft Defender for Endpoint through Microsoft Learn. The notes summarize key concepts, workflows, and practical knowledge gained during the learning path.

The goal of this repository is to document important security concepts, SOC workflows, investigation techniques, alert handling, endpoint protection, automation, and Kusto Query Language (KQL) used by Security Operations Center (SOC) analysts.

This repository focuses on learning the platform, understanding detection and response workflows, and preparing for real-world SOC environments.

## Learning Objectives

- Understand Microsoft Defender XDR architecture
- Learn Microsoft Defender for Endpoint capabilities
- Investigate alerts and incidents
- Understand automated investigation and response
- Learn endpoint onboarding concepts
- Practice Kusto Query Language (KQL)
- Understand SOC investigation workflow
- Prepare for Microsoft Applied Skills assessment
