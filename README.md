# 📖 Overview

This repository contains structured notes created while studying **Microsoft Defender XDR** and **Microsoft Defender for Endpoint** through the Microsoft Learn learning path.

The goal of this repository is to document the concepts, workflows, investigation techniques, alert management process, endpoint security capabilities, automation features, and Kusto Query Language (KQL) introduced throughout the course.

These notes are intended to serve as a personal reference while strengthening my understanding of Microsoft's security ecosystem and preparing for SOC Analyst roles.

---

# 🎯 Learning Objectives

- Understand Microsoft Defender XDR architecture
- Learn Microsoft Defender for Endpoint capabilities
- Investigate security alerts and incidents
- Understand automated investigation and remediation
- Learn endpoint onboarding concepts
- Explore Microsoft Defender security workflows
- Practice Kusto Query Language (KQL)
- Prepare for Microsoft Applied Skills assessment
